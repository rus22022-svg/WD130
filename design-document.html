Color Scheme

An editorial palette inspired by warm paper, charcoal ink, and a terracotta accent. Five core colors.

Background #faf7f2
Surface #f0ebe0
Dark / Text #1c1a17
Accent #c4603a
Secondary #4a7c6f

Font Choices

Two fonts imported from Google Fonts — a serif display and a clean humanist sans-serif.

Display Font

Playfair Display — Headlines & Accents

Used for: h1–h3, logo name, large numbers. Adds editorial character and elegance.

Body Font

DM Sans — Body & UI Text

Used for: paragraphs, nav links, labels, buttons. Clean and highly legible at small sizes.

Logo Design

A simple monogram mark — initials inside a circle — paired with name and subtitle text. Scales down to work in the footer and small contexts.

Required HTML Elements

Every required element and where it appears in the project.

<h1> – <h6> Used throughout all pages for page, section, and card headings
<p> Paragraph text in every section across all pages
<a> Nav links, card links, footer links, ICE external links (target=_blank)
<img> About page photo placeholder; ICE exercise screenshots
<ul> / <li> Nav menus, skill-list in hero card, fact lists on about page
<table> Course schedule table on index.html with thead, tbody, th, td
<form> Contact form on index.html with input, select, textarea, button
<header>, <nav>, <main>, <footer> Semantic landmark elements used on every page
